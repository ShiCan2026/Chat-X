## 准备工作

你需要准备以下物品：

- 一台可以上网的电脑（Windows系统，建议Windows 10/11）
- 足够的磁盘空间（至少500MB）
- 稳定的网络连接

## 安装Node.js环境

Node.js是运行后端服务的必要环境，我们需要先安装它：

### 步骤1：下载Node.js安装包

1. 打开浏览器，访问 [Node.js官网](https://nodejs.org/zh-cn/)
2. 点击页面上的"LTS"版本（长期支持版）下载按钮
3. 下载完成后，找到下载的文件（通常在"下载"文件夹中）

### 步骤2：安装Node.js

1. 双击下载的Node.js安装包
2. 出现安装界面后，点击"Next"（下一步）
3. 勾选"I accept the terms in the License Agreement"（我接受许可协议条款），然后点击"Next"
4. 选择安装位置（建议使用默认位置），点击"Next"
5. 在"Custom Setup"界面，点击"Next"
6. 点击"Install"（安装）开始安装
7. 安装完成后，点击"Finish"（完成）

### 步骤3：验证Node.js安装

1. 按下键盘上的 `Win + R` 组合键
2. 在弹出的窗口中输入 `cmd`，然后按回车键
3. 在命令行窗口中输入 `node -v`，然后按回车键
4. 如果显示版本号（如 v18.16.0），说明安装成功
5. 同样，输入 `npm -v` 验证npm是否安装成功

## 部署后端服务

后端服务是聊天应用的核心，我们需要先部署它：

### 步骤1：复制项目文件

1. 确保你已经有Chat-X的完整项目文件
2. 将项目文件夹复制到你想要安装的位置（例如：`D:\ChatX`）

### 步骤2：安装后端依赖

1. 按下键盘上的 `Win + R` 组合键
2. 输入 `cmd`，按回车键打开命令行窗口
3. 在命令行窗口中输入 `cd D:\ChatX\server`（注意：根据你的实际安装路径调整），然后按回车键
4. 输入 `npm install`，然后按回车键
5. 等待安装完成（可能需要几分钟，期间不要关闭窗口）

### 步骤3：启动后端服务

1. 在刚才的命令行窗口中，输入 `npm start`，然后按回车键
2. 如果看到类似 `Server is running on port 3000` 的提示，说明后端服务启动成功
3. 不要关闭这个命令行窗口，否则服务会停止

## 配置前端项目

前端项目需要连接到后端服务，我们需要修改一些配置：

### 步骤1：找到配置文件

1. 打开项目文件夹 `D:\ChatX\src`
2. 找到以下文件并使用记事本打开：
   - `pages/chat-password/chat-password.vue`
   - `pages/chat/chat.vue`
   - `pages/admin/admin.vue`

### 步骤2：修改服务器地址

在每个文件中，找到类似 `http://your-server-domain:3000` 的内容，将其替换为 `http://localhost:3000`（如果是在同一台电脑部署）。

例如：
```javascript
// 原代码
const serverUrl = 'http://your-server-domain:3000';

// 修改后
const serverUrl = 'http://localhost:3000';
```

## 打包前端应用

我们将使用HBuilderX来打包前端应用：

### 步骤1：安装HBuilderX

1. 打开浏览器，访问 [HBuilderX官网](https://www.dcloud.io/hbuilderx.html)
2. 点击"下载"按钮，选择"正式版"下载
3. 下载完成后，解压到你想要安装的位置（例如：`D:\HBuilderX`）

### 步骤2：导入项目

1. 打开HBuilderX
2. 点击"文件" -> "导入" -> "导入本地项目"
3. 选择 `D:\ChatX` 文件夹，点击"确定"

### 步骤3：打包为桌面应用

1. 在HBuilderX中，右键点击项目名称
2. 选择"发行" -> "原生App-云打包"
3. 选择"Windows"平台
4. 点击"发行"按钮
5. 等待打包完成，下载生成的安装包

### 步骤4：打包为移动端应用

1. 在HBuilderX中，右键点击项目名称
2. 选择"发行" -> "原生App-云打包"
3. 选择你需要的平台（Android/iOS）
4. 按照提示填写相关信息
5. 点击"发行"按钮
6. 等待打包完成，下载生成的安装包

## 启动和测试应用

### 步骤1：确保后端服务正在运行

- 检查之前打开的命令行窗口，确认 `Server is running on port 3000` 的提示仍然存在
- 如果服务已经停止，重新在 `D:\ChatX\server` 目录下运行 `npm start`

### 步骤2：安装并启动前端应用

1. 找到打包好的前端安装包
2. 双击安装包，按照提示安装应用
3. 安装完成后，打开应用
4. 应用应该能正常运行，你可以创建聊天室并发送消息

## 常见问题与解决方案

### 问题1：Node.js安装失败

**解决方案**：
- 确保下载的是适合你Windows系统的版本（32位或64位）
- 关闭所有杀毒软件和防火墙后重试
- 以管理员身份运行安装程序

### 问题2：后端服务启动失败

**解决方案**：
- 检查是否有其他程序占用了3000端口
- 重新运行 `npm install` 确保所有依赖都已正确安装
- 查看命令行窗口中的错误信息，根据提示解决

### 问题3：前端无法连接到后端

**解决方案**：
- 确保后端服务正在运行
- 检查前端配置文件中的服务器地址是否正确
- 确保防火墙没有阻止应用的网络连接

### 问题4：应用打包失败

**解决方案**：
- 确保HBuilderX是最新版本
- 检查网络连接是否正常
- 按照HBuilderX的提示填写完整的打包信息

## 进阶：部署到云服务器

如果你想让应用在互联网上可访问，可以考虑部署到云服务器：

1. 购买云服务器（推荐阿里云、腾讯云等）
2. 按照云服务商的指南配置服务器
3. 在服务器上重复上述部署步骤
4. 配置域名和SSL证书（可选）
